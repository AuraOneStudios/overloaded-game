# Security Policy

## Supported Versions

Currently, only the `main` branch and the latest published version of **OVERLOADED** are supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |
| Older   | :x:                |

## Reporting a Vulnerability

We take the security of OVERLOADED seriously. If you discover a security vulnerability within this project, please **do not** open a public issue.

Instead, please send an email to `security@auraonestudios.com` or use the GitHub Security Advisory private reporting feature if enabled on this repository.

Please include:
- A detailed description of the vulnerability.
- Steps to reproduce the issue.
- Potential impact.

We will acknowledge receipt of your vulnerability report within 48 hours and strive to send you regular updates about our progress. If you report a vulnerability, please allow us reasonable time to release a patch before disclosing it publicly.
